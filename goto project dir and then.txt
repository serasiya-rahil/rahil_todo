goto project dir and then

1. pip install virtualenv
2. create venv using: virtualenv venv
3. activate venv: cd venv/Scripts; Activate
4. install django using: pip install django
5. cd ../../
6. create a django project: django-admin startproject ToDo .
7. create and app using: django-admin startapp app
	NOTE: app in above command is the appname we can replace it anywhere

8. Run the project: python manage.py runserver
9. copy urls.py(from ToDo folder) to app folder
10. create a folder inside the app folder 
11.python manage.py migrate -> db creation and admin user
12.python manage.py createsuperuser
